# 🌱 IoT-Assisted Automated Plant Watering System

## 📖 Overview
The IoT-Assisted Automated Plant Watering System is a smart irrigation solution developed using the ESP8266 Wi-Fi development board. The system continuously monitors soil moisture and automatically controls a water pump whenever the soil becomes dry. Sensor data is transmitted to the ThingSpeak cloud platform, enabling real-time monitoring and visualization from anywhere with an internet connection.

## 🎯 Objectives
- Automate plant irrigation based on soil moisture levels.
- Minimize water wastage through intelligent irrigation.
- Enable real-time monitoring using ThingSpeak Cloud.
- Reduce manual intervention in plant care.

## ⚙️ Working Principle
1. The soil moisture sensor continuously measures the moisture level of the soil.
2. The ESP8266 processes the sensor data.
3. When the moisture level falls below the predefined threshold, the relay module activates the water pump.
4. The pump supplies water until the desired moisture level is reached.
5. The ESP8266 uploads the sensor readings to the ThingSpeak cloud platform for real-time monitoring and data visualization.
